# Spring Boot Hands-On Projects

A collection of hands-on Spring Boot projects demonstrating real-world backend development concepts, best practices, and enterprise-grade application design using Java.

This repository contains a curated set of hands-on projects built using **Spring Boot**, aimed at strengthening practical backend development skills with **Java**.

Each project focuses on real-world use cases and demonstrates industry-standard practices such as layered architecture, RESTful API design, database integration, exception handling, security, and performance optimization.

## Key Concepts Covered
- Spring Boot fundamentals & auto-configuration
- REST API development (CRUD, pagination, filtering)
- Spring Data JPA & Hibernate
- Database integration (MySQL / PostgreSQL / H2)
- Exception handling & global error responses
- Validation & DTO mapping
- Spring Security (JWT, role-based access)
- Logging & monitoring
- Testing (JUnit, Mockito)
- Clean code & best practices

## Purpose
- Build production-ready backend services
- Gain hands-on experience with Spring Boot
- Prepare for backend / full-stack interviews
- Showcase real-world projects on GitHub & CV

## Tech Stack
- Java 8+ / Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- Hibernate
- Maven / Gradle
- MySQL / H2
- JUnit & Mockito
